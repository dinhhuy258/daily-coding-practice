Given an array A, of N integers A.

Return the highest product possible by multiplying 3 numbers from the array.

**Note::**

Solution will fit in a 32-bit signed integer.

**Example 1:**
```
 Input:
 A = [1, 2, 3, 4]

 Output:
 24

 Explanation:
 2 * 3 * 4 = 24
```

**Example 2:**
```
 Input:
 A = [0, -1, 3, 100, 70, 50]

 Output:
 350000

 Explanation:
 70 * 50 * 100 = 350000
```
